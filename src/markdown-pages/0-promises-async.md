# Test bericht

An async function is a function declared with the `async`keyword**.**Async functions are instances of the[`AsyncFunction`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/AsyncFunction)constructor, and the`await`keyword is permitted within them. The`async`and`await`keywords enable asynchronous, promise-based behavior to be written in a cleaner style, avoiding the need to explicitly configure promise chains.

Async functions may also be defined[as expressions](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/async_function).

## Syntax

async function

### Parameters

`name`

The function’s name.

`param`

The name of an argument to be passed to the function.

`statements`

The statements comprising the body of the function.  The`await`mechanism may be used.
